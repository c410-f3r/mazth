# mazth
A simple implementation of matrix related utilities. 

## Features
### Matrices sizes ( 2x1, 3x1, 4x1, 1x2, 1x3, 1x4, 3x3, 4x4 )
### Matrix utiliies ( 3x3 cross product, etc )
### Matrix types ( for f32, f64 )
### Quaterion
### Dual Quaternion
### Quaternion utiliies ( slerp, sclerp )
